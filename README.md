# Cognitive Neuroscience Projects | Spring 2025

This repository contains computational projects from the Spring 2025 Master's course in Cognitive Neuroscience.

## Quick Overview

| Project | Key Focus | Tools |
| :--- | :--- | :--- |
| **Psychophysics** | Modeling face recognition performance. | Python |
| **Electrophysiology** | Spike sorting & neural variability analysis. | MATLAB, ROSS |
| **EEG RSVP** | Visual processing & multivariate pattern analysis. | MATLAB, EEGLAB |
| **SleepEEGNet** | Deep learning for automated sleep staging. | Python |

---

## Brief Details

*   **Psychophysics:** Psychometric curve fitting and AIC-based model comparison for face recognition stimuli.
*   **Electrophysiology:** Spike detection using ROSS, followed by MMFF-based neural variability and stimulus decoding analysis.
*   **EEG RSVP:** Analysis of EEG data during visual tasks, covering preprocessing, ERPs, MVPA, and RSA.
*   **SleepEEGNet:** Deep learning architecture for sleep stage classification using Sleep-EDF data and SMOTE.
